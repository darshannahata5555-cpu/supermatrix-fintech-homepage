# SuperMatrix fintech homepage

A responsive static homepage for SuperMatrix, focused on simple mutual fund investing, SIP tools, and goal-based wealth planning.

## Local preview

Serve the repository with any static web server and open `index.html`.

```powershell
python -m http.server 8000
```

Then visit <http://localhost:8000>.

## Deployment

The site is deployed through GitHub Pages from the repository's `main` branch.
